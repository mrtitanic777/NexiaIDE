                        Partial SDK Installation Detected
                        
Your SDK is missing include/ and lib/ directories needed for compilation. This happens when the SDK was installed without Visual Studio 2010.

                        
Nexia can fix this by temporarily adding registry keys so the SDK installer allows a Full Installation. This only touches Visual Studio 2010 application keys — it cannot affect Windows stability or boot.

                    

                    

                    
                    
                    

                    
After preparing, uninstall then reinstall the Xbox 360 SDK and select "Full Installation".
